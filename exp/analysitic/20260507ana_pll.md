## 统一图像结构

step1 / step2 / step2s 使用同一套读图方式。Kuramoto 的图像结构不同，在后面的 Kuramoto 小节单独说明。

Case 1 是无噪声 baseline，直接分析系统矩阵 `A`，因此图中有五行：

1. 相图轨迹。
2. 奇异值谱。
3. SVD 向量载荷。
4. 特征值谱。
5. 特征向量载荷。

Case 2 到 Case 5 分析后向矩阵：

```text
B = A^T Sigma^{-1} A
```

当 `Sigma` 正定时，`B` 是实对称半正定矩阵，所以奇异值和特征值应当一致，SVD 左奇异向量和特征向量只差符号或重根子空间内旋转。因此这些图只保留三行：

1. 相图轨迹。
2. 奇异值谱。
3. 左奇异向量载荷。

图标题中的 `p` 表示 `process_noise`，只控制相图中的轨迹递推噪声；`a/b` 或 `sigma_a/sigma_b` 控制矩阵分析中的 `Sigma`。

## Step1

step1 系统为：

```text
x_{k+1} = lambda x_k
y_{k+1} = mu y_k + (lambda^2 - mu) x_k^2
psi = [x, y, x^2]^T
```

对应观测层矩阵：

```text
A = [[lambda, 0, 0],
     [0, mu, lambda^2 - mu],
     [0, 0, lambda^2]]
```

噪声协方差为：

```text
Sigma = [[a, 0, b],
         [0, a, 0],
         [b, 0, a]]
```

合法参数满足：

```text
a > b >= 0
```

### Step1 参数设计

- Case 1：`a=b=0`，直接分析 `A`。13 组 `lambda/mu` 参考点拆成两张 baseline 图。
- Case 2：固定每组 `lambda/mu`，令 `b=0`，扫描 `a = 0.01, 0.1, 1, 10, 100, 1000`。
- Case 3：固定 `lambda/mu/a`，扫描 `b/a = 0, 0.1, 0.3, 0.6, 0.85, 0.98`。
- Case 4：固定 `lambda/a/b`，扫描 `mu`。
- Case 5：固定 `mu/a/b`，扫描 `lambda`。

### Step1 结果理解

Case 1 中，`A` 通常不是对称矩阵，所以 SVD 和 EVD 看的是两种不同结构。EVD 的特征值直接对应 `lambda, mu, lambda^2`，而 SVD 会额外反映非正交耦合项 `lambda^2 - mu` 带来的放大方向。因此不能期待右奇异向量和特征向量一致。

Case 2 中，`b=0` 时：

```text
Sigma = aI
B = (1/a) A^T A
```

这意味着改变 `a` 主要只是把谱整体按 `1/a` 缩放，奇异向量方向基本不变。CSV 检查也支持这一点：对每张 Case 2 图计算 `sv1 * a`，同一图内的相对波动最大只有约 `4.7e-14`，属于数值误差级别。第一行相图变“更散”主要来自 `process_noise` 同步增大，而不是来自 `Sigma` 本身。

Case 3 中，`b/a` 越接近 1，`Sigma` 越接近奇异，后向矩阵里的某些方向会被显著放大，向量载荷更容易旋转。这一类图适合看“噪声结构”如何改变宏观方向。

Case 4 和 Case 5 中，系统动力学本身改变。一个重要参照点是：

```text
lambda^2 = mu
```

此时 `A` 中的耦合项 `lambda^2 - mu` 消失，`y` 与 `x^2` 的混合减弱；偏离该关系越多，SVD 更容易看到由非正交耦合造成的主方向变化。

step1 的谱范围很宽，`sv1` 在当前扫描中从 `1e-05` 到约 `3.20e7`，说明大参数组合下的后向放大非常强。读图时不宜只比较柱子绝对高度，还要结合每个面板自己的 log y 轴。

## Step2

step2 系统为：

```text
x_{k+1} = rho x_k + kappa y_k^2
y_{k+1} = rho y_k
psi = [x, y, y^2]^T
```

对应观测层矩阵：

```text
A = [[rho, 0, kappa],
     [0, rho, 0],
     [0, 0, rho^2]]
```

这里：

```text
rho   = 原 step2 里的 a，表示线性收缩/增长率
kappa = 原 step2 里的 coupling，表示 y^2 -> x 的耦合强度
```

step2 使用的 Sigma 为：

```text
Sigma = [[sigma_a, 0,       0      ],
         [0,       sigma_a, sigma_b],
         [0,       sigma_b, sigma_a]]
```

即 `sigma_coupling = y_y2`。

### Step2 参数设计

- Case 1：`sigma_a=sigma_b=0`，直接分析 `A`。
- Case 2：固定 `rho/kappa`，令 `sigma_b=0`，扫描 `sigma_a`。
- Case 3：固定 `rho/kappa/sigma_a`，扫描 `sigma_b/sigma_a`。
- Case 4：固定 `rho/sigma_a/sigma_b`，扫描 `kappa`。
- Case 5：固定 `kappa/sigma_a/sigma_b`，扫描 `rho`。

### Step2 结果理解

step2 的一个关键点是：`kappa` 不改变 `A` 的特征值，但会强烈改变奇异值。因为 `A` 是上三角矩阵，它的特征值只由对角线决定：

```text
eig(A) = rho, rho, rho^2
```

但 `kappa` 出现在 `y^2 -> x` 的非正交耦合项中，会明显改变 SVD 看到的主伸缩方向。因此在 Case 1 中，EVD 更像是在看局部线性增长率，SVD 更能捕捉 `kappa` 造成的剪切/非正交放大。

Case 2 与 step1 一样，`sigma_b=0` 时：

```text
Sigma = sigma_a I
B = (1/sigma_a) A^T A
```

CSV 检查显示同一张 Case 2 图内 `sv1 * sigma_a` 的相对波动最大约 `3.7e-13`，仍然只是数值误差。因此 Case 2 主要读“谱整体缩放”和“相图递推噪声 p 的可视化影响”，不要把它误读成方向变化。

Case 3 中，`sigma_b` 耦合的是 `y` 与 `y^2` 这两个输出噪声坐标。它会改变后向矩阵，但与动力学里的主耦合 `y^2 -> x` 不是同一对坐标。因此在很多大 `kappa` 组合里，它对最大奇异值的放大相对温和。

## Step2s

step2s 使用同一个 step2 动力学和同一批参数，只改变 Sigma 结构。它暂时命名为 `step2s`，表示 step2 的普通 Sigma 版本。

step2s 的 Sigma 为：

```text
Sigma = [[sigma_a, 0,       sigma_b],
         [0,       sigma_a, 0      ],
         [sigma_b, 0,       sigma_a]]
```

也就是 `sigma_coupling = x_y2`。这和 step1 中的普通 Sigma 结构一致：对角线有 `sigma_a`，左下角与右上角有 `sigma_b`。

输出目录：

```text
exp/analysitic/figs/ana_pll2s/
```

### Step2s 与 Step2 的对比

因为 Case 1 不使用 Sigma，Case 2 中 `sigma_b=0`，所以 `step2` 和 `step2s` 在 Case 1 与 Case 2 完全一致。

差异从 `sigma_b > 0` 开始出现，主要集中在 Case 3，其次是 Case 4 和 Case 5。按 `step2s` 相对 `step2` 的奇异值相对变化统计：

```text
Case 1: sv1 中位相对变化 0，最大 0
Case 2: sv1 中位相对变化 0，最大 0
Case 3: sv1 中位相对变化约 9.6%，最大约 2425%
Case 4: sv1 中位相对变化约 3.2%，最大约 9.7%
Case 5: sv1 中位相对变化约 2.4%，最大约 18.9%
```

最大的差异出现在 `sigma_b/sigma_a = 0.98` 且 `kappa` 很大的组合。例如：

```text
rho = 0.05
kappa = 100
sigma_a = 100
sigma_b = 98
```

此时：

```text
step2  的 sv1 约为 100.00
step2s 的 sv1 约为 2525.13
```

也就是说，普通 Sigma 结构会把这一组的最大后向奇异值放大约 25 倍。

这个差异有一个清楚解释：step2 的动力学耦合是 `y^2 -> x`，而 step2s 的 Sigma 正好耦合 `x` 与 `y^2` 噪声坐标。当 `sigma_b` 接近 `sigma_a` 时，Sigma 接近奇异，且这个近奇异方向和动力学耦合方向更对齐，所以 `A^T Sigma^{-1} A` 会出现更强的放大。相比之下，step2 的 `y_y2` Sigma 耦合没有直接对齐 `y^2 -> x` 这个动力学通道，所以放大通常弱一些。

## Kuramoto

第四部分使用 `kuramoto` 文件夹里的聚类 Kuramoto 数据生成思路，并参考 `kuramoto_exp_gisce` 中用时间配对拟合线性算子 `A` 的方式。

默认设置为：

```text
N = 10
n_clusters = 2
T = 100
dt = 0.01
noise = 0
random_seed1 = 41
random_seed2 = 2600
burn_in_steps = 1000
sample_stride = 10
lag_steps = 1
```

数据先由相位转为 `cos(theta), sin(theta)` 状态，再构造 identity + 一阶 Fourier observable。默认 `N=10` 时，原始状态维度是 20，observable 维度是 60。

本部分暂时不引入噪声矩阵扫描，默认：

```text
Sigma = I
B = A^T A
```

因此第二行和第三行分别读 `A^T A` 的奇异值谱与左奇异向量。第一行不再画原始轨迹，而是画整体序参量和两个团内序参量随时间的变化。

Kuramoto 的第二行目前保留“每列独立 y 轴”，因为不同参数下 `A^T A` 的谱量级差异可能很大。为了避免误读，第二行每一列都显示自己的 y tick，并标注 `s(A^T A)`。


参数筛选原则是：保留 `K_intra > K_inter`，并且让整体序参量和两个团内序参量明显不同。当前 60 个组合里：

```text
r_group1_mean / r_group2_mean 的最小值约 0.9694
r_total_mean 范围约 0.5966 到 0.9212
order_gap 范围约 0.0786 到 0.3813
```

其中 `order_gap = 0.5 * (r_group1_mean + r_group2_mean) - r_total_mean`，它越大，说明“团内同步强，但全局同步弱”的结构越明显。

### Kuramoto Case 1

Case 1 固定团内耦合，但固定值本身也取多组：

```text
K_intra = 2, 4, 6, 10, 16, 20
K_inter = 0, 0.02, 0.05, 0.1, 0.2
```

这一组全部满足 `K_intra > K_inter`，用来观察在团内耦合占优时，逐渐增大团间耦合会如何削弱“两团各自同步但全局不同步”的结构。

弱团间耦合区域最符合当前想看的前提。以 `K_intra=6` 为例，`K_inter = 0.05` 和 `0.1` 时，两个团内已经几乎同步：

```text
r_group1_mean ≈ 0.9986
r_group2_mean ≈ 0.9969
```

但整体序参量仍然低很多：

```text
K_inter = 0.05: r_total_mean ≈ 0.6320, order_gap ≈ 0.3657
K_inter = 0.10: r_total_mean ≈ 0.6392, order_gap ≈ 0.3585
```

这说明系统不是“没有同步”，而是更接近“两团分别同步，但两团之间还没有锁到同一个全局相位”。当 `K_inter` 增大到 `0.2` 时，整体序参量升到约 `0.896` 到 `0.921`，`order_gap` 降到约 `0.079` 到 `0.085`；它还没有和团内序参量完全重合，但全局同步已经明显增强。

谱上看，`sv1(A^T A)` 并不是随 `K_inter` 单调变化：

```text
K_intra = 2:  sv1 范围约 3.18 到 589.16
K_intra = 6:  sv1 范围约 8.28 到 1146.42
K_intra = 20: sv1 范围约 53.90 到 5844.32
```

这个非单调形状提示 `A^T A` 对弱团间耦合区的相对相位方向很敏感。尤其当 `K_inter` 在 `0.05` 或 `0.1` 附近时，序参量仍显示两团未全局锁定，但谱上可能出现很强主方向。

### Kuramoto Case 2

Case 2 固定团间耦合，但固定值本身也取多组：

```text
K_inter = 0, 0.02, 0.05, 0.1, 0.2
K_intra = 2, 4, 6, 10, 16, 20
```

这一组也全部满足 `K_intra > K_inter`。和上一版不同，这里不再用 `K_inter=4`，因为那会让整体序参量和团内序参量几乎重合，无法突出团结构。现在固定的团间耦合都很弱，因此整体序参量和团内序参量保持明显分离：

```text
K_inter <= 0.1: order_gap 约 0.35 到 0.38
K_inter = 0.2: order_gap 约 0.079 到 0.085
```

因此 Case 2 现在更适合读“在弱团间耦合下，增强团内耦合会怎样改变主谱方向”。当 `K_inter=0` 时，`r_total_mean` 只从约 `0.623` 到 `0.637`，团内序参量却从约 `0.970` 以上继续接近 1；当 `K_inter=0.2` 时，整体序参量升到约 `0.896` 到 `0.921`，说明团间耦合已经开始显著拉近两团。

谱上，`sv1(A^T A)` 对 `K_intra` 的增强很敏感。例如：

```text
K_inter = 0:    sv1 约 3.18 -> 759.94
K_inter = 0.05: sv1 约 110.12 -> 2673.21
K_inter = 0.10: sv1 约 109.03 -> 5844.32
K_inter = 0.20: sv1 约 3.78 -> 53.90
```

这说明当前参数区间里，谱强度不只由“是否全局同步”决定，还会被弱团间耦合下的相对相位动力学显著影响。

从 `case_summary.csv` 里的载荷统计看，除少数弱团间过渡列外，第一奇异向量的绝对载荷主要集中在一阶 cos Fourier 块，identity 和 sin 块通常很小。这和 Kuramoto 相位系统的周期结构是一致的：用 Fourier observable 拟合时，主方向不是原始坐标本身，而是周期嵌入中的方向。

## 当前最有用的结论

1. step1 / step2 / step2s 的 Case 2 主要是“谱缩放实验”。当交叉噪声为 0 时，改变 `a` 或 `sigma_a` 不会真正改变方向结构。

2. `step2` 的 SVD/EVD 差异比 step1 更值得强调。因为 `kappa` 不进入特征值，但强烈进入奇异值，所以 EVD 会低估非正交耦合带来的主放大方向。

3. `step2s` 是必要对照。它说明同一个 step2 动力学下，仅仅改变 Sigma 的交叉项位置，就可能让后向谱发生数量级变化。尤其当 `sigma_b/sigma_a` 接近 1 时，普通 Sigma 的 `x-y^2` 交叉项会显著增强 `y^2 -> x` 通道。

4. Kuramoto 现在的参数更聚焦“团内同步强、整体同步弱”的区域。Case 1 看不同 `K_intra` 下弱团间耦合的影响；Case 2 看不同弱 `K_inter` 下增强 `K_intra` 如何改变 `A^T A` 的主方向。

5. 后续如果要讨论“噪声结构如何选择宏观方向”，Case 3 是最核心的读图对象；如果要讨论“动力学参数如何改变主方向”，则应重点读 Case 4 和 Case 5。

6. 图里的相图和矩阵谱要分开理解。相图里的扩散由 `process_noise = p` 控制；矩阵谱和载荷由 `Sigma` 与 `A` 决定。二者放在同一列是为了辅助解释，但不是同一个噪声来源。
